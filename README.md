# Telepítés
1. Keresd ki a megfelelő kiadást és töltsd le a `verzió.zip` fájl-t.
2. Keresd meg a játékot `Steam/steamapps/common/SimRail`.
3. Csomagold ki a SimRail mappába a letöltött zip-et.
